# 📌 标签书签管理器

一个Chrome浏览器插件，用标签管理浏览器书签，让书签分类更灵活高效。

## ✨ 功能特性

1. **新标签页显示** - 打开新标签页时自动显示插件主页
2. **书签展示** - 返显浏览器中的所有书签
3. **两栏布局** - 左栏显示标签，右栏显示书签列表
4. **标签筛选** - 点击左侧标签快速筛选对应的书签
5. **分页显示** - 书签列表支持分页（每页12个）
6. **灵活标签** - 在书签名字后添加 `#标签名` 来为书签添加标签
9. **新增标签** - 鼠标悬停书签卡片时可直接输入新标签并回车添加
7. **自动提取** - 自动从书签名字中提取并识别标签
8. **搜索功能** - 可搜索书签名称、标签和URL
9. **实时更新** - 书签变化时自动刷新

## 🚀 安装方式

### 方法一：本地加载（开发者模式）
1. 打开 Chrome 浏览器
2. 在地址栏输入 `chrome://extensions/`
3. 启用右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择本插件的文件夹
6. 完成！

## 📖 使用指南

### 添加标签到书签

在书签名称后添加 `#标签名`，可以添加多个标签，例如：

- `Google #搜索 #工具`
- `GitHub #开发 #代码`
- `MDN文档 #编程 #参考`

### 查看书签

1. 打开新标签页，插件主页自动显示
2. 左侧显示所有标签及其对应的书签数量
3. 点击"全部书签"查看所有书签
4. 点击具体标签筛选该分类的书签

### 搜索书签

在顶部搜索框输入：
- 书签名称的关键字
- 标签名称
- 网址关键字

搜索会实时过滤匹配的书签。

### 标签丸操作

在书签卡片中，每个标签都显示为标签丸（#tag）：
- 点击标签丸可直接筛选该标签的书签
- 悬停时会高亮显示

### 分页导航

- 书签列表默认每页显示12个
- 页码按钮位于列表底部
- 支持上一页、下一页、页码直跳

## 💡 使用建议

### 标签命名规范

建议使用简短、统一的标签名：
- **分类标签**: `开发`, `设计`, `工作`, `学习`
- **功能标签**: `搜索`, `工具`, `文档`, `参考`

### 添加新标签

在任意书签卡片上悬停，将出现一个小输入框：
1. 输入标签名称（支持自动提示当前标签）
2. 按 Enter 或点击空白处保存
3. 新标签会被追加到书签名称并同步至浏览器

- **平台标签**: `Google`, `GitHub`, `AWS`, `阿里`

### 最佳实践

1. **统一风格** - 对同类书签使用相同的标签
2. **多标签组合** - 为同一个书签添加多个标签
 - ✅ 支持标签多选与交集筛选
 - ✅ 新增书签卡片标签输入功能
3. **定期清理** - 删除不需要的书签时自动更新标签
4. **命名简洁** - 标签名保持简短，便于搜索

## 🐛 常见问题

**Q: 如何删除书签上的标签？**  
A: 编辑书签，在标题中删除对应的 `#标签` 部分保存即可。

**Q: 能否将一个书签添加到多个标签？**  
A: 可以！在书签名称后添加多个标签，如：`标题 #标签1 #标签2`

**Q: 修改书签后多久会更新显示？**  
A: 通常在1秒内自动更新。

**Q: 搜索功能支持模糊匹配吗？**  
A: 支持，搜索会匹配书签名称、标签和URL中的关键字。
