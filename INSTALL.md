# ⚡ 快速安装指南

## 📥 安装步骤

### 第1步：打开扩展程序管理页面
1. 打开 Chrome 浏览器
2. 在地址栏输入：`chrome://extensions/`
3. 按回车键

### 第2步：启用开发者模式
- 点击右上角的「**开发者模式**」开关（右侧会显示已启用）

### 第3步：加载扩展程序
1. 点击左上角的「**加载已解压的扩展程序**」按钮
2. 在弹出的文件夹选择对话框中，选择本插件的文件夹
3. 点击「选择」

### 第4步：完成安装 ✅
- 你应该能看到「标签书签管理器」已安装
- 如果一切正常，Chrome 会显示一个确认信息

## 🚀 测试插件

### 打开新标签页
- 在 Chrome 中按 `Ctrl+T`（`Cmd+T` 在Mac上）打开新标签页
- 你应该能看到插件的主页面，显示所有书签

### 查看调试信息（可选）
如果遇到问题，可以查看开发者日志：
1. 回到 `chrome://extensions/` 页面
2. 找到「标签书签管理器」
3. 点击「**检查时的错误**」查看任何错误信息

## 📚 为书签添加标签

### 编辑现有书签
1. 按 `Ctrl+Shift+B`（`Cmd+Shift+B` 在Mac上）打开书签栏
2. 右键点击任何书签 → 点击「**编辑**」
3. 在名称字段后添加标签，例如：`Google #搜索 #工具`
4. 点击「**保存**」

### 示例
```
GitHub #开发 #代码           ← 带两个标签的书签
MDN Web Docs #文档 #前端    ← 带两个标签的书签
YouTube                       ← 无标签的书签
```

## ✨ 开始使用

打开新标签页，你会看到：
- 👈 **左侧**：所有标签列表
- 👉 **右侧**：书签网格显示
- 🔍 **顶部**：搜索框（可搜索书签名称、标签或URL）

点击任何标签快速筛选，或在搜索框输入关键字！

## 🔄 如何更新插件

如果你修改了插件代码：
1. 回到 `chrome://extensions/` 页面
2. 找到「标签书签管理器」
3. 点击左下角的「**刷新**」按钮
4. 新标签页会显示更新后的内容

## ❌ 卸载插件

1. 回到 `chrome://extensions/` 页面
2. 找到「标签书签管理器」
3. 点击右下角的「**移除**」按钮

## 🆘 常见问题排查

### 问题1：看不到书签
**解决方案：**
1. 检查你的Chrome中是否真的有书签
2. 检查浏览器控制台是否有错误（F12 → 控制台选项卡）
3. 尝试刷新新标签页（Ctrl+Shift+R强制刷新）

### 问题2：标签没有显示
**解决方案：**
1. 检查书签名称是否包含 `#` 符号
2. 标签格式必须为：`书签名 #标签` （#号后面不能有空格）
3. 确保书签已正确保存

### 问题3：插件不显示在新标签页
**解决方案：**
1. 检查 `manifest.json` 中 `chrome_url_overrides` 设置
2. 尝试卸载后重新安装
3. 打开 F12 检查控制台是否有错误消息
