## 🚀 3分钟快速开始

### 1. 安装插件（2分钟）
```
1. Chrome 地址栏输入: chrome://extensions/
2. 右上角启用: 开发者模式
3. 点击: 加载已解压的扩展程序
4. 选择: 本项目文件夹
5. 完成! ✅
```

### 2. 添加书签标签（1分钟）
```
1. Ctrl+Shift+B 打开书签管理器
2. 编辑书签，在名称后加: #标签名
   例如: Google #搜索 #工具
3. 保存
```

### 3. 打开新标签页查看效果
```
Ctrl+T → 看到插件主页 ✅
```

## 💡 标签格式速查

```javascript
// ✅ 正确格式
"Google #搜索"              // 1个标签
"GitHub #开发 #代码"        // 2个标签
"MDN #文档 #前端 #参考"     // 3个标签

// ❌ 错误格式
"Google #搜索 #"            // #后没有标签名
"GitHub# 代码"              // #前有空格
"Test# tag"                 // #后有空格（只识别 #tag）
```

## 🔍 搜索能力

在顶部搜索框输入：

```
输入内容           → 搜索范围
"GitHub"          → 书签标题
```

## 📋 故障排除快查表

| 问题 | 解决方案 |
|------|---------|
| 看不到书签 | 检查是否有权限 / 刷新 F5 / 检查F12控制台 |
| 标签没显示 | 检查书签名是否含 `#` / 确保格式 `#tag` |
| 搜索无结果 | 检查搜索词拼写 / 清空搜索框重试 |
| 分页不工作 | 检查书签数 > 12 / 清空过滤条件 |
| UI错乱 | `Ctrl+Shift+R` 强制刷新 / 重启Chrome |
| 权限错误 | 检查 `manifest.json` / 重新加载扩展 |

## 🎓 关键概念

**标签（Tag）**: 在书签名称后加 `#` 前缀，用于分类
```
📌 Book Name #tag1 #tag2
          ↑ 用#号标记
```

**标签丸（Tag Pill）**: UI中显示的标签按钮
```
[#开发] [#前端]  ← 这些就是标签丸
```

**过滤**: 点击标签后只显示含该标签的书签
```
点击 #开发 → 只显示含 #开发 标签的书签
```

**分页**: 每页显示12个书签，超过时分页
```
第1页: 书签1-12
第2页: 书签13-24
...
```
